Estas son las diferentes rutas para poder usar la Api Rest.

    Ver datos
  • http://localhost:4000/viewData
  • http://localhost:4000/viewData/usuarios
  • http://localhost:4000/viewData/usuarios/:id
    • --------------------------------
    Crear datos
  • http://localhost:4000/crearData
  • http://localhost:4000/crearData/usuarios
    • --------------------------------
    Modificar data
  • http://localhost:4000/modificarData/:id
    • --------------------------------
    Eliminar data
  • http://localhost:4000/eliminarData/Trabajadores/:id
  • http://localhost:4000/eliminarData/Usuarios/:id
